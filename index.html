M/M/s Queueing Model

core formulas
◆ Model Formulation

Utilization (ρ)

ρ = λ / (s × μ)

Must be < 1 for stability.

Empty Prob (P₀)

P₀ = 1 / [Σ(λ/μ)n/n! + (λ/μ)s/(s!(1-ρ))]

Queue Length (Lq)

Lq = P₀ × (λ/μ)s × ρ / [s!(1-ρ)2]

System Length (L)

L = Lq + λ/μ

Queue Time (Wq)

Wq = Lq / λ (Little's Law)

System Time (W)

W = Wq + 1/μ
Assumptions: Poisson arrivals, exponential service times, infinite queue, FCFS discipline, s identical servers.
